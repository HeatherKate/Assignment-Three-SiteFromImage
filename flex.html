The JPEG Image

The term "JPEG" is an acronym for the Joint Photographic Experts Group, which created the standard. The MIME media type for JPEG is image/lpeg, except in older Internet Explorer versions, which provides a MIME type of image/pjpeg when unloading JPEG images. JPEG files usually have a filename extension of .jpg or .jpeg.

JPEG/JFIF supports a maximum image size of 65,535 x 65,535 pixels, hence up to 4 gigapixels (for an aspect ratio 1:1).
